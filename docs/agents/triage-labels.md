# Triage 标签

技能使用五个标准 triage 角色。本文件将这些角色映射到本仓库 issue tracker 中实际使用的标签字符串。

| 角色 | 标签 | 含义 |
|------|------|------|
| `needs-triage` | `needs-triage` | 维护者需要评估此 issue |
| `needs-info` | `needs-info` | 等待提报人提供更多信息 |
| `ready-for-agent` | `ready-for-agent` | 需求明确，AI Agent 可直接接手 |
| `ready-for-human` | `ready-for-human` | 需要人类实现 |
| `wontfix` | `wontfix` | 不处理 |

当技能提到某个角色时（如"打上 AFK-ready 的 triage 标签"），使用上表中对应的标签字符串。

如需修改标签名称，编辑右侧列即可。
